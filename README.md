# 什麼是皮下組織壓力損傷

褥瘡 :

- 是皮膚受到壓迫或摩擦, 導致皮膚受傷甚至深及皮下組織，肌肉與骨頭的情形。皮膚溫度升高也會加速褥瘡的形成。

- 好發於骨頭突出的部位

- 併發症有蜂窩性組織炎，骨髓炎，敗血症等

- 在美國每年有250萬人得褥瘡相關的疾病，每年總共花費90 到 110億美金醫療費，每年有6萬人死於褥瘡的併發症
